# Personal Finance Visualizer

Track your income and expenses easily!

## Features

- Add/Edit/Delete Transactions
- MongoDB for storage
- Clean UI with Tailwind
- Next.js (App Router) + API Routes

## Tech Stack

- Next.js 14
- Tailwind CSS
- MongoDB Atlas
- Deployed on Vercel
